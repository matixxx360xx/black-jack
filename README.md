## üÉè Key Features
The game adheres strictly to casino-standard Black Jack rules:

1.  **Smart Ace Handling:** Ace values (`A`) automatically toggle between **11 and 1** to prevent the player from going Bust (exceeding 21).
2.  **Dealer Logic:** The dealer automatically draws cards until their total reaches **17** or more (`Stand on 17`).
3.  **Dynamic Turn Control:** Functional **"Karta"** (Hit) and **"Pasuj"** (Stand) buttons, with correct reveal of the Dealer's hidden card upon standing.
4.  **Deck Management:** Deck generation, shuffling, and card drawing functions are encapsulated within the `TaliaKart` class.
